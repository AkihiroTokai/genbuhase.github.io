More Records
1. Overview このModはMinecraft PEにレコードを追加します。
2. How to Use 最初にこのModを導入すると、Mojangフォルダー内にMore Records/Records.Jsonが生成されます。 生成されたRecords.Jsonを、必ず下記の形式に沿って編集して下さい。 入力した内容に応じて追加されるレコードの情報が変化します。 
[
	{
		"ID": レコードID(int型),
		"Name": レコードの名前(String型),
		"Type": レコードの色(1～12),
		"URL": 音声ファイルの絶対パス(String型)
	},

	{
		...
	}
]
